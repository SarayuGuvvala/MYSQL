# ✈️ Airline Reservation & Operations Analytics System

## Project Overview

The **Airline Reservation & Operations Analytics System** is a relational database project developed using **MySQL**. This project is designed to manage airline operations such as maintaining passenger information, flight schedules, airport details, aircraft information, bookings, payments, crew assignments, and operational analytics.

The database demonstrates the implementation of:
- Database creation
- Table creation
- Primary and foreign keys
- Data insertion
- Relationships between tables
- SQL queries of different difficulty levels

---

## Technologies Used

- MySQL
- SQL

---

## Database Tables

The project contains the following tables:

1. Passengers
2. Airports
3. Aircraft
4. Flights
5. Bookings
6. Payments
7. Crew
8. Operations_Analytics

---

## Relationships

- One passenger can make multiple bookings.
- One flight can have multiple bookings.
- One aircraft can operate multiple flights.
- One aircraft can have multiple crew members assigned.
- One airport can serve as the source or destination for multiple flights.
- Each booking is associated with one payment.
- Each flight generates one operational analytics report.
